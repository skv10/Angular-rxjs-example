Of Operator

The of operator in RxJS is a creation operator that allows you to create an Observable from one or more values. It is a simple way to emit a fixed number of synchronous values in sequence and then complete.

Features

  • Emits the provided values sequentially, one at a time.
  • Completes immediately after emitting all values.
  • Values can be of any type.
