Concat Operator

In RxJS, the concat function is a combination operator that takes multiple Observables as input and concatenates them sequentially,emitting values from one Observable only after the previous one has completed. This is particularly useful when you want to process streams of data in a strict sequence

Features

  • Sequential Execution: Observables are executed one after another. The next Observable starts only after the current one completes.
  • Order Preservation: The values are emitted in the order of the input Observables.
  • Handles Completion: If an Observable does not complete, subsequent Observables will not start.

Example

const obs1 = of('A', 'B').pipe(delay(1000)); // Emits after 1 second

const obs2 = of('C', 'D');

concat(obs1, obs2).subscribe(res => console.log(res));
